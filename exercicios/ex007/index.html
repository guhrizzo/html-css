Principais formatações

Negrito / Destaque

Nessa frase temos um termo em Negrito usando a tag <B> (não semantica)

Nessa frase tenho um termo em Destaque graças a tag <STRONG> (semantica)

Itálico / Ênfase

Nesta frase há um termo em itálico atráves da tag <I> (Não semantica)

Nesta frase temos um termo em ênfase usando a tag <EM> (semantica)

Texto Marcado

Podemos colocar um texto marcado usando a tag <MARK>

Texto grande e pequeno

Estamos criando um texto grande e um testo pequeno nesse paragrafo usando as tags <BIG> e <SMALL>

Texto Deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido porém não considerado usando a tag <DEL>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <INS>

porém podemos usar também a tag <U> (Não semantica)

Texto subrescrito

Para inserir coisas como: x20+3 usamos a tag <SUP>

Texto Subscrito

Para inserir coisas do tipo: H2O usamos a tag <SUB>
